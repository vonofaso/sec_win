## Курсовой проект: «Развертывание и обеспечение безопасности доменной инфраструктуры Windows»

**Цель проекта:**  
Освоение практических навыков построения защищенной корпоративной сети на базе Microsoft Windows Server и клиентских ОС.

**Задачи и выполненные работы:**
- Развертывание полной доменной инфраструктуры (AD DS, DNS, DHCP) на Windows Server.
- Настройка сетевой безопасности и разграничения доступа к ресурсам (файловый сервер, ACL NTFS).
- Автоматизация установки клиентских ОС по сети (WDS / MDT) и их ввод в домен.
- Централизованное управление конфигурациями и политиками безопасности через Group Policy (GP).
- Внедрение антивирусной защиты в сетевой среде (Kaspersky Security Center, Endpoint Security).
- Настройка резервного копирования (Windows Backup) для обеспечения целостности и восстановления данных.

**Результат:**  
Спроектирована и развернута эталонная доменная среда с базовым уровнем защищенности. Получены навыки системного администрирования и понимание принципов обеспечения информационной безопасности на уровне инфраструктуры.
